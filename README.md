# HumanClock

## How to Run Rest API

### 1. Clone this repository
### 2. Make sure you are using Java 11 and Maven 3.x
### 3. You can build the project and run the tests by running mvn clean package
### 4. You can use raise any get request and can see the JSON response. (Request URL : GET | http://localhost:8080/clock/getTime?args=23:30)
