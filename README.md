# HumanClock

## Objective 1 & 2 can be verified by Objective1_2.java, whereas HumanClock is a Spring Boot Rest API for Objective 3.

## How to Run Rest API

### 1. Clone this repository
### 2. Make sure you are using Java 11 and Maven 3.x
### 3. You can build the project and run the tests by running mvn clean package
### 4. You can raise any get request and can see the JSON response. (Request URL : GET | http://localhost:8080/clock/getTime?args=23:30)
